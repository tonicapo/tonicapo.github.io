Ecran :

Résolution physique de l'écran.

Commande Javascript : screen.width et screen.height

Ces commandes doivent normalement retourner la résolution d'affichage de l'écran en pixels physiques (et non en dips). Cependant, cela varie en fonction des navigateurs et des périphériques.

Avec Chrome 25 sur un galaxy Note, les valeurs retournées sont exprimées en dips, mais pas avec Firefox.

Pour information : px = dips x (dpi / 160)

Navigateur :

Commande Javascript : window.outerWidth et window.outerHeight

Dimensions totales du navigateur, incluant les barres d'outils, ascenceurs, etc.

Viewport :

Commande Javascript : window.innerWidth et window.innerHeight

Dimensions du "viewport visuel" (Visual viewport).

Il s'agit des dimensions, en pixels CSS, du contenu affiché sur la page. Ces valeurs peuvent dépendre du périphérique utilisé, du navigateur, etc.

Layout :

Commande Javascript : document.documentElement.clientWidth et document.documentElement.clientWidth

Dimensions de la page à afficher (Layout viewport). Il s'agit des dimensions, en pixels CSS, de la page complète. Cette valeur est constante, quel que soit le niveau de zoom.

Le rapport entre les valeurs de Visual viewport et de Layout viewport est égal à 1 lorsque le niveau de zoom est minimum.

On peut le forcer Layout Viewport à avoir une largeur égale à celle du Visual viewport en utilisant la balise meta :

<meta name="viewport" content="width=device-width">

Densité de pixels :

Commande Javascript : window.devicePixelRatio

Densité de pixels du périphérique.
